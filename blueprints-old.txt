{% block page_title %}Terraform blueprints available for {{gcp_name_short}}{% endblock %}

{% block body %}

A blueprint is a package of deployable, reusable configuration and policy that
implements and documents a specific opinionated solution. For more details see,
[blueprints overview](/anthos-config-management/docs/concepts/blueprints).

This page provides a list of [blueprints/modules](#blueprints) that are packaged as
Terraform modules and can be used for creating resources for {{gcp_name}}.
There are also two end-to-end [examples](#end-to-end-blueprints) available.

## End-to-end blueprints {: id="end-to-end-blueprints" }

These end-to-end examples are designed to be forked as a starting point. These
examples are not suitable for direct usage in production scenarios:

- [Example Foundation](https://github.com/terraform-google-modules/terraform-example-foundation){: .external} - Example repository showing how the CFT modules can be composed to build a secure cloud foundation.
- [CFT Fabric](https://github.com/terraform-google-modules/cloud-foundation-fabric){: .external} - This repository includes an advanced example designed for prototyping.

* [Confidential data warehouse](https://cloud.google.com/architecture/confidential-data-warehouse-blueprint){: .external}

* [Protecting confidential data in AI](https://cloud.google.com/architecture/protecting-confidential-data-in-ai-platform-notebooks){: .external}
